## Search Operator Cheat Sheet

### Google Search Operators

```
BASIC:
"phrase"                    Exact phrase
-word                       Exclude word
*                          Wildcard
..                         Number range
OR                         Either/or

SITE:
site:domain.com            Within domain
site:domain.com/path       Within path
site:.edu                  Domain extension
-site:domain.com           Exclude domain

CONTENT:
intitle:keyword            In title
allintitle:keyword         All in title
inurl:keyword              In URL
allinurl:keyword           All in URL
intext:keyword             In text
allintext:keyword          All in text

FILES:
filetype:pdf               PDF files
filetype:xlsx              Excel
filetype:docx              Word
filetype:pptx              PowerPoint

ADVANCED:
cache:url                  Cached version
related:url                Related pages
link:url                   Pages linking to
define:word                Definition
weather:city               Weather
stocks:symbol              Stock info
```

```

to:user                    To specific user
```

## Python Scripts for OSINT

### Basic Web Scraper

```python
import requests
from bs4 import BeautifulSoup
import csv

def scrape_sponsors(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.content, 'html.parser')
    
    sponsors = []
    for sponsor in soup.find_all('div', class_='sponsor'):
        name = sponsor.find('h3').text
        logo = sponsor.find('img')['src']
        sponsors.append({'name': name, 'logo': logo})
    
    return sponsors

def save_to_csv(data, filename):
    with open(filename, 'w', newline='') as f:
        writer = csv.DictWriter(f, fieldnames=data[0].keys())
        writer.writeheader()
        writer.writerows(data)

# Usage
sponsors = scrape_sponsors('https://twincitiestartupweek.com/sponsors')
save_to_csv(sponsors, 'sponsors.csv')
```

## Troubleshooting

### Common Issues

**Problem**: Website blocked/rate limited
**Solution**: Use VPN, add delays, use different IP

**Problem**: Information not found
**Solution**: Try different search terms, check archives, look at related sources

**Problem**: Conflicting information
**Solution**: Check dates, verify sources, note discrepancy

**Problem**: Data format inconsistency
**Solution**: Standardize in spreadsheet, use data cleaning tools

**Problem**: Too much data
**Solution**: Prioritize by relevance, create filtering system, use database
