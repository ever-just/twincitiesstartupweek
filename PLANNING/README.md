# PLANNING — SMS × TCSW 2026

Live operating system for the SMS × BETA partnership to produce Twin Cities Startup Week 2026 (September 14–18).

Every file in this folder answers one of four questions:

1. **What do we know?**
2. **What do we believe?**
3. **What have we decided?**
4. **What must happen next?**

## Structure

```
PLANNING/
├── thesis/            # Core argument: why SMS, why now, what we're building
├── stakeholders/      # SMS team, BETA contacts, key relationships
├── operating-model/   # BETA-SMS partnership structure, roles, decision rights
├── venues/            # Venue pipeline, location architecture, capacity
├── funding/           # Budget model, sponsor pipeline, funding strategy, offers
├── meetings/          # Board-chair prep, Angela/Linda sessions, operating cadence
└── deliverables/      # Decision briefs, proposals, consumption-layer plans
```

## Historical context

Research, OSINT, sponsor history, session archives, and press coverage are in `BACKGROUND/`.
