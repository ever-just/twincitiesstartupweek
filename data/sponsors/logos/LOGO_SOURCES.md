# Sponsor Logo Sources
*Where to download official logos for each confirmed TCSW sponsor*

## HOW TO DOWNLOAD
1. Visit the Brand/Press URL listed below
2. Download SVG or PNG (prefer SVG for scalability)
3. Save as: `[COMPANY_NAME]_logo.[ext]` in this folder
4. Note the license/usage restrictions in the table
