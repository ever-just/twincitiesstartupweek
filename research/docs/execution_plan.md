## Notes & Tips

### Research Tips
- Take screenshots of important pages
- Save URLs for all sources
- Note access dates
- Document any conflicts
- Keep detailed notes
- Follow leads and connections
- Don't skip verification

### Organization Tips
- Use consistent naming
- Create clear folder structure
- Use templates for data entry
- Back up data regularly
- Version control important files
- Create index/table of contents
- Document methodology

### Time Management
- Set daily goals
- Track progress
- Take breaks
- Don't rush verification
- Batch similar tasks
- Use templates to save time
- Automate where possible

---

## Contingencies

**If behind schedule:**
- Extend timeline by 1-2 weeks
- Focus on high-priority data first
- Use AI agents to accelerate research
- Reduce scope if necessary
- Prioritize current/recent data

**If data not found:**
- Try alternative search terms
- Check related sources
- Use Wayback Machine
- Contact organizations directly
- Note as gap in analysis
- Recommend for future research

**If overwhelmed:**
- Break into smaller chunks
- Focus on one topic at a time
- Use templates and checklists
- Delegate if possible
- Take breaks
- Adjust timeline
