# TCSW Runtime OS

This folder is the live operating system for SMS x BETA's proposed end-to-end production and community experience partnership for Twin Cities Startup Week 2026.

## Purpose

This is where SMS converts research, meeting discovery, strategy, and execution into durable operating artifacts.

The repo is now being separated into two layers:

1. `00-weldon-research/` — preserved historical intelligence and source research.
2. `01-runtime-os/` — live 2026 execution system.

## Runtime OS rule

Every file in this folder should answer one of four questions:

1. **What do we know?**
2. **What do we believe?**
3. **What have we decided?**
4. **What must happen next?**

## Initial operating sequence

1. Package the existing historical research into `00-weldon-research/`.
2. Create the current discovery synthesis from the SMS/Angela conversation.
3. Create the Monday board-chair meeting prep.
4. Create the SMS x BETA end-to-end production proposal.
5. Build out funding, operating model, venues, programming, brand/marketing, and product workstreams.
